# Waqar Asghar — Tutor portfolio

This is a lightweight single-page portfolio built for easy deployment to GitHub Pages.

## What is included

- `index.html` — single-page markup
- `styles.css` — CSS variables and responsive styles
- `script.js` — theme toggle, timeline renderer, and contact form handler
- `data/timeline.json` — your teaching history and enriched entries
- `assets/` — avatar and CV placeholder

## Customize

- Update contact details in `index.html` and `assets/Waqar_Asghar_CV.txt`.
- If you use Formspree, set the endpoint in `script.js` at the `endpoint` variable.

## Deploy to GitHub Pages

1. Commit and push to `main`.
2. In GitHub repo settings, enable Pages: deploy from `main` branch -> save.
3. The site will be available at `https://<your-username>.github.io/<repo>`.
